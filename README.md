# Сайт «Воронежский клуб байдарочников»

Многостраничный сайт клуба. Дизайн — по образцу сайта клуба «Крокус», палитра заменена на «воду» (тёмно-синий/бирюзовый + белый + закатный оранжевый акцент), альпинизм → байдарки, горы → реки.

## Структура

```
index.html      — Главная
routes.html     — Маршруты (карта рек + маркеры километража)
classes.html    — Занятия (расписание)
gallery.html    — Фотогалерея
news.html       — Новости
contacts.html   — Контакты (карта базы)
apply.html      — Заявка (форма вместо анкеты)
assets/
  styles.css    — все стили
  script.js     — карта, маршруты, меню, форма
  logo.png      — ваш логотип (уже вставлен)
build.py        — генератор страниц (по желанию, для правок сразу во всех страницах)
```

## Как открыть
Просто откройте `index.html` в браузере. Для карт и фото нужен интернет.
Карты — **Яндекс.Карты** (нужен бесплатный API-ключ, см. ниже).

## 📝 Как редактировать сайт (просто)

Сайт — это обычные файлы, которые открываются и правятся в любом текстовом
редакторе (Блокнот, VS Code, Notepad++). Никакой установки не нужно.

**Изменить текст** (заголовки, описания, цены, новости):
1. Откройте нужную страницу (`.html`) в текстовом редакторе.
2. Найдите нужный текст между тегами (например, `<h1>...</h1>` или `<p>...</p>`)
   и просто перепишите его.
3. Сохраните файл и обновите страницу в браузере.

**Где что менять:**

| Что изменить | Где |
|---|---|
| Тексты, заголовки, цены | прямо в `.html`-файле нужного раздела |
| Логотип | замените файл `assets/logo.png` (тем же именем) |
| Фотографии в галерее | `gallery.html` — ссылки на картинки |
| Новости | `news.html` — блоки `<article class="news-card...">` |
| Цвета сайта | `assets/styles.css` — вверху блок `:root` (переменные цвета) |
| Телефон, адрес, контакты | `contacts.html` + низ любой страницы (подвал) |
| Маршруты и километраж на карте | `assets/script.js`, объект `routesData` |

> ⚠️ Важно: файл `build.py` — это вспомогательный генератор (я им собирал
> страницы). **Он вам не нужен для правок.** Редактируйте `.html`-файлы напрямую —
> изменения сразу видны. Запускать `build.py` не требуется.

Хотите править сайт «мышкой» без кода — скажите, подключим бесплатную
визуальную CMS (например, Netlify CMS / Tilda-подобный редактор).

## Карты (Яндекс) — нужен бесплатный ключ
Карты маршрутов и контактов работают на Яндекс.Картах. Чтобы они отобразились:
1. Зайдите на https://developer.tech.yandex.ru/services/ и войдите.
2. «Подключить API» → выберите **«JavaScript API»** (Геокодер НЕ нужен — мы используем готовые координаты) → «Продолжить». Получите ключ (бесплатно).
3. Откройте `assets/script.js`, найдите строку `const YANDEX_API_KEY = "ВАШ_КЛЮЧ_ЯНДЕКС_КАРТ";`
   и вставьте свой ключ вместо `ВАШ_КЛЮЧ_ЯНДЕКС_КАРТ`.

Пока ключ не вставлен, вместо карты показывается аккуратная подсказка (сайт не ломается).

## Как выложить в интернет (как «Крокус» на GitHub Pages)
1. Создайте репозиторий на GitHub, загрузите все файлы (index.html, остальные .html, папку assets).
2. Settings → Pages → Source: ветка `main`, папка `/root`.
3. Через минуту сайт будет доступен по ссылке вида `https://ВАШ-ЛОГИН.github.io/имя-репозитория/`.

## Что нужно заменить на реальные данные
- **Маршруты и километраж:** файл `assets/script.js`, объект `routesData` — координаты точек и число км в поле `km`. Маркеры на карте строятся автоматически.
- **Карта базы клуба:** `assets/script.js`, переменная `CLUB_COORDS` (координаты базы).
- **Фото:** ссылки на Unsplash в `.html` — замените на реальные снимки клуба.
- **Контакты:** телефон, почта, адрес, ссылки на ВК/Telegram — в шапке/подвале и на `contacts.html`.
- **Форма заявки:** сейчас это макет (показывает «спасибо», но никуда не отправляет).
  Чтобы заявки приходили вам, подключите бесплатный сервис (например, Formspree)
  или свой обработчик — по желанию помогу настроить.

## Логотип
Ваш логотип уже вставлен (`assets/logo.png`, круглый, фон прозрачный, стоит по центру
на главной и в шапке/подвале). Чтобы заменить — положите новый файл с тем же именем
`assets/logo.png` (лучше квадратный PNG с прозрачным фоном).
