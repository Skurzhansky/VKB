# Сайт «Воронежский клуб байдарочников»

Многостраничный сайт клуба. Дизайн — по образцу сайта клуба «Крокус», палитра заменена на «воду» (тёмно-синий/бирюзовый + белый + закатный оранжевый акцент), альпинизм → байдарки, горы → реки.

## Структура

```
index.html      — Главная
routes.html     — Маршруты (карта рек + маркеры километража)
classes.html    — Занятия (расписание)
gallery.html    — Фотогалерея
news.html       — Новости
contacts.html   — Контакты (карта базы)
apply.html      — Заявка (форма вместо анкеты)
assets/
  styles.css    — все стили
  script.js     — карта, маршруты, меню, форма
  logo.png      — ваш логотип (уже вставлен)
build.py        — генератор страниц (по желанию, для правок сразу во всех страницах)
```

## Как открыть
Просто откройте `index.html` в браузере. Для карт и фото нужен интернет
(подключены Leaflet, OpenStreetMap и фотографии).

## Как выложить в интернет (как «Крокус» на GitHub Pages)
1. Создайте репозиторий на GitHub, загрузите все файлы (index.html, остальные .html, папку assets).
2. Settings → Pages → Source: ветка `main`, папка `/root`.
3. Через минуту сайт будет доступен по ссылке вида `https://ВАШ-ЛОГИН.github.io/имя-репозитория/`.

## Что нужно заменить на реальные данные
- **Маршруты и километраж:** файл `assets/script.js`, объект `routesData` — координаты точек и число км в поле `km`. Маркеры на карте строятся автоматически.
- **Карта базы клуба:** `assets/script.js`, переменная `clubLatLng` (координаты базы).
- **Фото:** ссылки на Unsplash в `.html` — замените на реальные снимки клуба.
- **Контакты:** телефон, почта, адрес, ссылки на ВК/Telegram — в шапке/подвале и на `contacts.html`.
- **Форма заявки:** сейчас это макет (показывает «спасибо», но никуда не отправляет).
  Чтобы заявки приходили вам, подключите бесплатный сервис (например, Formspree)
  или свой обработчик — по желанию помогу настроить.
